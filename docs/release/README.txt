PicPack v1.0.0 README
======================

PicPack は、画像を作成日時順に並べ、指定した枚数ごとに指定したフォルダ数へ振り分ける Windows 用ツールです。
差分画像、投稿用画像セット、素材整理などで、同じ枚数ずつ複数フォルダに分けたいときに使えます。


起動方法
--------
インストールは不要です。
ZIP を解凍して、PicPack.exe をダブルクリックして起動してください。

想定環境:
- Windows 10
- Windows 11


基本的な使い方
--------------
1. 「入力フォルダを選択」で、整理したい画像が入っているフォルダを選びます。
2. 「出力フォルダを選択」で、振り分け先のフォルダを選びます。
3. 「1フォルダあたりの枚数」を指定します。
4. 「使用するフォルダ数」を指定します。
5. 「コピーする」または「移動する」を選びます。
6. 実行前確認の内容を確認してから実行します。

初めて使うときは、元画像が残る「コピーする」で試すことをおすすめします。


対象画像形式
------------
PicPack が対象にする画像形式は以下です。

- jpg
- jpeg
- png
- webp
- bmp

入力フォルダ直下の画像が対象です。サブフォルダ内の画像は対象にしません。


並び順について
--------------
画像は Windows 上のファイル作成日時順に並び替えます。
これは撮影日時ではありません。

ファイルをコピー、ダウンロード、展開したタイミングによって、Windows のファイル作成日時が変わることがあります。
思った順番にならない場合は、ファイルの作成日時を確認してください。


振り分け仕様
------------
PicPack は、指定フォルダ数まで使ったら最初のフォルダへ戻るラウンドロビン方式で振り分けます。

例:
1フォルダあたりの枚数 = 3
使用するフォルダ数 = 7

画像001 -> 001
画像002 -> 001
画像003 -> 001

画像004 -> 002
画像005 -> 002
画像006 -> 002

画像007 -> 003
画像008 -> 003
画像009 -> 003

画像010 -> 004
画像011 -> 004
画像012 -> 004

画像013 -> 005
画像014 -> 005
画像015 -> 005

画像016 -> 006
画像017 -> 006
画像018 -> 006

画像019 -> 007
画像020 -> 007
画像021 -> 007

画像022 -> 001
画像023 -> 001
画像024 -> 001

つまり、111,222,333,444,555,666,777,111,222... のように振り分けます。


コピーと移動
------------
コピー:
元の画像を残したまま、出力フォルダへコピーします。

移動:
元の画像を出力フォルダへ移動します。

移動モードでは元画像の場所が変わります。移動モードを使う前に、必ずバックアップを取ることをおすすめします。


同名ファイルについて
--------------------
出力先に同名ファイルがある場合、既存ファイルは上書きしません。
image.png が既にある場合は image_001.png、image_002.png のように連番を付けます。


設定の保存
----------
「1フォルダあたりの枚数」と「使用するフォルダ数」は次回起動時にも復元されます。


アンインストール
----------------
展開した PicPack のファイルを削除すればアンインストールできます。
設定も完全に削除したい場合は、以下のフォルダも削除してください。

%LOCALAPPDATA%\PicPack


注意
----
- 最初はコピーで動作を確認してください。
- 移動モードを使う前にバックアップを取ってください。
- 本ソフトウェアは無保証です。
- 大切な画像を扱う前に、少量のコピー画像で動作確認してください。
